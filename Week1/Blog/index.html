ASP.NET Core ile İlk Projem: Kütüphane Yönetim Sistemi

ASP.NET Core MVC kullanarak bir kütüphane yönetim sistemi geliştirdim. Bu proje, kullanıcı kimlik doğrulamadan veri yönetimine kadar birçok özelliği içeriyor. Entity Framework Core ve LINQ ile veritabanı işlemlerini optimize ettim.

  • Ödünç alma işlemleri için CRUD operasyonları uygulandı.
  • Kullanıcılar kitapları ekleyip silebilir.
  • Bootstrap ile kullanıcı dostu bir arayüz tasarlandı.

Daha fazla bilgi için ASP.NET Core dokümantasyonunu inceleyebilir veya projemin GitHub reposunu ziyaret edebilirsiniz.

ReactJS ile Dinamik Arayüzler Geliştirme

ReactJS ile İnsan Kaynakları Yönetim Sistemi’nin frontend’ini geliştirdim. Bu proje, dinamik ve hızlı arayüzler oluşturmanın gücünü gösterdi. Component-based yapı ile kod yeniden kullanılabilir hale geldi.

  1. Component’ler oluşturarak modüler bir yapı kurdum.
  2. State yönetimi için React Hooks kullandım.
  3. API entegrasyonu için Axios ile veri çektim.

React öğrenmek için React’in resmi sitesini ziyaret edebilir veya projemin GitHub reposunu inceleyebilirsiniz.

Semantik HTML: Web Sayfalarını Daha Anlamlı Yapma

Semantik HTML, web sayfalarını daha erişilebilir ve anlaşılır yapar.`main`, `section`, `article` gibi etiketler kullanarak özgeçmiş ve blog sayfamı oluşturdum. Bu, ekran okuyucular ve SEO için büyük bir avantaj.

  • `main`: Sayfanın ana içeriğini tanımlar.
  • `section`: Tematik bölümleri ayırır.
  • `article`: Bağımsız içerikler için idealdir.

Daha fazla bilgi için MDN Web Docs sayfasını veya W3C HTML5 Semantik kılavuzunu inceleyebilirsiniz.
