## İçerik

- **Sayfa Başlığı:** Sınıf Tablosu
- **Tablo Alanları:**
  - Öğrenci Numarası
  - Adı
  - Soyadı
  - Notu

- **Öğrenci Sayısı:** 

## Kullanılan Teknolojiler

- HTML5
- CSS3 (Inline `<style>` kullanılarak)

- Tablo genişliği %100 ayarlanmıştır.
